### Usage
    1 . Activate The Virtual Env
        - `cd py_env/Scripts/`
        - `activate.bat`

    2.  Set the STAGE Environment Value     
        in .env file. It can be `development` or `production`

    3.  Execute main.py 
        `python main.py`
